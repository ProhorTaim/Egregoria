# Egregoria 0.7.0

## Основные изменения

- UI и окно настроек стабилизированы.
- Исправлена работа сохранения/чтения пользовательских настроек.
- Добавлено переключение языка (English/Russian).
- Добавлен режим пониженного рендер-разрешения на macOS (Retina) через настройку `Low resolution mode`.
- Улучшено поведение полноэкранного режима при старте игры.
- Улучшено терраформирование.
- Исправлены баги строительства дорог на высоте.

## Производительность

- Для macOS добавлена опция понижения фактического рендер-разрешения (требуется перезапуск).
- Рекомендуется запускать игру только в `--release` режиме.

## Платформы релиза

- Windows
- macOS
- Linux
